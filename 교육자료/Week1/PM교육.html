◎PM교육내용

※노트정리 내용

· 입회 시 고객먼저 데리고 내려가면 안된다.[생산관리들이 싫어한다]

· ReBuyoff는 시트가지고 확인만 한다.

· 스크랩은 3개까지, 5개 쌓이면 답이 없다.

· ☆릴리프(Relief) → 간섭 생길경우 or 스크랩 쌓일경우(안 떨어질 때) / 도려내는것, 털어내는것 = 도피

· 챔퍼 이유? = 안다치게 하기 위해

· 잡아주는 것 = 패드 / (트림스틸) ≠ 치는 것 = 펀치 [후 공정 부터 있다 / 2차 부터]

· 블랭크 홀더 > 다이페이스 → [1차에만 있다]

· 리테이너(Retainer) = 유지/보정 장치

· 음각 = 상형 / 양각 = 하형

· 용접자국이 없어야 하는데 라인이 두꺼워서 글라인더가 안들어 가는 경우도 있다.

· 녹 제거를 위해 금형을 뒤집어 놓는 경우도 있다.

· 스크랩 슈트 = 스크랩 취출을 도와주는 것[내려가도록]

· Stain = Welding spot - 사상량과 관련이 있다.

· GEOMETRY[검사구] = CF(checking fixture) = IF(Inspection fixture)

· DFPP = 제품변경 → Data 수정 요청 보고서

· PDM = 시방서

· FRC = 클램프에 위치시키는 점보 데이터(클램프 유무에 따라 점수차이↑)

· 패턴도 → 검사 → 출하 = 하루

· TGA = 가공데이터(To Go Ahead)

· ATL = 초도(All tool)

· DFO = 주물 투입 가능 시점

· 공법검도 = P.A = Process Agreement

· DDR(Die Design Review) = 구상도

· 패턴검사 = 패턴이 잘 만들어졌는지 확인 하는것, 제작되어있는 것을 검사하러 가는것 → 실제 금형 크기와 같다.

· 패턴검사중 붙어있는 노란색 테이프 = 수작업 해야 할 부분

· 후공정 = 데이터 + 10 / 드로우는 필요 X

· 패턴 = Pattern(Poly)

· 주물 = Casting

· 면삭 = 2D Machining

· 형상가공 = 3D Machining

·※패턴 투입 = 패턴 제작 시작 = 패턴 출도 = ERP상의 "패턴도"

·※패턴 출하 = 주물 투입 = 주물제작 시작 = ERP상의 패턴 출하

· Undercut = 작업이 안되는 부분

· SET-UP = 순서대로 X, Y, Z 좌표에 놓는 것 → 좌표 바뀌는 경우 = 작업성↑

· A필러, B필러, C필러 = 자동차 사이드 아우터에 들어가는 필러들 [맨앞, 중간, 후면부]

· ※DDM = 가공 기준홀

· 경사면 = rake face / slope face

· Over hang(빨간색 색칠된것) 빼고 → Pad 제거 = 암묵적 Rule

· SPM = 1분에 생산하는 샘플량

· 충돌 / 간섭 = Collision

· 르노에서 DP = Devant Presse(Front Press) → F = flow = 진행 방향

· 금형 사이즈는 mm단위 이다.

· Inventory No. / Marticule No. = 공정별 부여

· Part No. = 파트별 부여

· 명판 = Name Plate

· 인판넬 = 들어오는것 → 로딩판넬

· 아웃판넬 → 성형된 상태

· JIG 용 = 덜 정밀 / CAR 용 = 더 정밀

· 샘플종류(명칭) → 르노 = IOD(초도) : 첫번째 샘플 / 지오메트리 타겟 X → VC1 VC2 .... → PTI = 마지막 샘플

· 동적검사(Dynamic) = 금형이 작동할 때 확인하는것

· 정적검사(Static) = 컴퓨터에 프로그램을 켜놓고 하는 것 / 금형 까놓고 한다

· 성형해석

· 재질이 가진 인장강도와 마찰

· 정보 = 프로젝트가 시작 할 때 받는다.

· Binder = Blank Holder

· 1.1 = 안전수치(여분의 힘) → 프레스를 계속 사용하다 보면 힘이 약해질수도 있고 여러가지 변수가 있다.<성형력>

· 제품률 = 단가 → 블랭크(소재)대비 생산되는 파트 무게 = 소재 x 파트무게

· Material Flow = 소재유입

· Formability = 성형성 → 크랙이 나는지 안나는지?

· Yield ratio = 돈 관련

· 초록색 = Good / 빨간색 = Red

· 대부분 접히는 부분의 Thinning 얇으면 부러질 위험이 있다

· 성적서 FIR - GEOMETRY

· 해석 예상 = 성적서와 동일하게

· Restrike = 성형 한번 더 하는 경우 주름, 크랙 등을 더 본다.

· Spring back = 판넬이 눌렸다 돌아오는 성질 → 더 눌려지면 Red / 튀어나오면 Blue [여러가지 조건을 가지고 해석]

· 라인을 못 만들 경우? → 구조·공법적으로 불리하거나 불가능한 경우? → 이런식으로 바꿔보자고 작성

· 검토구조? = 다른 자동차 파트들과 모두 결부 된다.

· Excel 사진 압축하여 파일 크기를 줄여 보내야 한다.

· 성형의 깊이에 따라 INR 찢어질 수도 있다.

· CP를 기준으로 패턴도를 만들고 패턴도과 완성되는 기준으로 패턴을 내보낸다.

· 일정은 금형의 크기에 따라 다르다.

· 스틸 = NC데이터 필요, 후 공정의 상형 = NC데이터 기준 + 10일

· TGA(가공데이터) → CP다음에 보낸다.

· TGA 나오고 나서 7 일 정도 후 1 2 3차 각각 7일 씩 소요

· ATL(All Tool) - 가장 중요 = ATL은 IOD보다 앞이다.

· 비행기는 운송이 일주일 정도 걸린다.

· 배는 운송이 8주정도 소요된다.

· B1 ~ B4 = 육성 돌리는 것

· FOB = 선적

· 내부적으로 일정을 타이트하게 짜서 Master plan을 작성해야한다.

· 비슷한 파트 = 비슷하게 일정 진행한다.

· 10 → 18日 / 15 → 21日 처럼 큰 파트는 일정을 이런식으로 수정하여 많이 줘야 한다.

· Master Plan 엑셀 파일 수정하는 법 → 제작일정관리 → 수정 → 모든파일로 변경*(항상 모든 파일로 바꿔야 파일이 뜬다.)

· ※Master Plan은 패턴도를 기준으로 작성한다 → 설계팀에서 받고 진행

· ※마스터 플랜에는 정확한 옳고 그름이 없다. 날짜는 금형의 크기 또는 변수에 의해 달라진다. → 금형의크기가 좌우 많이한다.

· 이젝터 핀 = 스크랩 자를때 취출을 도와주는 것

· MDC = 검사구

· 데꼬보꼬 = Uneven
